Assignment 2 - Dictionary search algorithm

BUILDING ----------------

To build the assignment simply type 'make', or 'make all'

to build with debug headers type 'make debug'

to delete the current build, type 'make clean'

to run some test cases type 'make run'

USAGE -------------------

type ./ok word

where word is the word you are looking for in the dictionary.

if it is found the program prints yes, otherwise it prints no.

DESCRIPTION -------------

ok.c contains the code which uses binary search method to search 

through a dictionary file. The file is read using read and lseek.
